1. Create Database
   su postgres;

   create table()
   insert record

2. files -> computer -> local -> lib ->open terminal
3. Check java version(java -version)
4. vim ~/.bashrc
5. In that file paste(export CLASSPATH=$CLASSPATH :.:/usr/local/lib/postgres-42.6.0.jar)
6. source ~/.bashrc
7. vim sample.java
8. javac -cp/usr/local/lib/postgresql-42.6.0.jar sample.java
9. java -cp .:/usr/local/lib/postgresql-42.6.0.jar sample.java
